Drafty is a webapp for use on mobile to facilitate smoother and quicker drafts!

Similar to other Magic companion apps, like Lifetap, Drafty sits in the middle of the table showing a dedicated region for each player. Rather than tracking life totals and dealing damage, Drafty acts like a chess clock, counting down each draft pick "turn" and showing which way to pass. This helps speed up drafts and synchronises passing to avoid the dreaded stack of packs.

Give it a go at https://blytht.github.io/drafty/
